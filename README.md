# 🐎 Bitácora de Rodeo

> Plataforma SaaS modular para la gestión de trabajo en terreno.

Bitácora de Rodeo es una plataforma en desarrollo orientada a la gestión de operativos en terreno. Su arquitectura modular permite administrar distintos tipos de actividades mediante módulos especializados, compartiendo un mismo núcleo de aplicación.

Actualmente el proyecto se encuentra en desarrollo activo y utiliza una arquitectura basada en HTML, CSS y JavaScript Vanilla, con Google Apps Script y Google Sheets como backend temporal. La persistencia de datos migrará progresivamente a Supabase.

---

# 🎯 Objetivo

Construir una plataforma SaaS que permita registrar, organizar y gestionar operativos de trabajo en terreno de forma simple, rápida y completamente digital.

El objetivo es reemplazar planillas manuales y registros en papel por una solución escalable, preparada para múltiples áreas de trabajo.

# 📌 Filosofía del proyecto

Bitácora de Rodeo no fue concebido como una aplicación para un único cliente.

Su diseño busca construir una plataforma modular donde diferentes áreas de trabajo puedan compartir una misma arquitectura y una misma experiencia de usuario.

Cada módulo representa un dominio específico, mientras que el núcleo de la aplicación permanece común para todos.
